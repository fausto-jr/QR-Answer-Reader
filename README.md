# QR-Bubble-Answer-Reader
Pyton OCR - To read a buble aswersheet and QR info

Dependencies
imutils
cv2
pdf2image

#How it works 
Read a PDF file can be several pages
Read QR info with personal data
Read Questions Answers
Save to a csv file.
